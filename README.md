# Bike Ride Weather Checker

This is simple web app that displays current weather conditions to help you decide if it's good biking weather. 

## Features

- Uses browser geolocation to get your location
- Displays temperature, chance of precipitation, and wind conditions
- Fetches data from the Open-Meteo API

## How to Run

1. Clone or download this repository
2. Run locally in browser
3. Allow location access when prompted
4. Click a button to see weather data
